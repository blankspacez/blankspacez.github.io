    # Vue项目开发流程

    浏览器加载 index.html 页面，发现 <div id="app"></div> 作为 Vue 应用的挂载点。

    main.js 中通过 createApp(App) 创建 Vue 应用，并通过 mount('#app') 将 App.vue 组件挂载到 #app 上。

    App.vue 渲染其模板内容，并将最终的 UI 渲染到 index.html 中的 <div id="app"></div> 元素。

    • index.html：提供 HTML 结构和挂载点。
    • main.js：入口文件，初始化 Vue 应用，创建并挂载根组件 App.vue。
    • App.vue：根组件，包含应用的布局和业务逻辑，作为 Vue 应用的起始组件。

    # API风格

    # 选项式API

    和在html代码中引入Vue模块的代码风格类似

    # 组合式API

    更灵活，实际开发中更推荐

    • setup：一个标识，告诉 Vue 需要进行一些处理，让我们可以更简洁的使用组合式 API。
    • ref()：接收一个内部值，返回一个响应式的 ref 对象，此对象只有一个指向内部值的属性 value。
    • onMounted()：在组合式 API 中的钩子方法，注册一个回调函数，在组件挂载完成后执行。

    Example

    • 1 创建Api.vue实现一个点击自增的button
    <script setup>
    import {onMounted, ref} from 'vue'
    const count = ref(0)

    function increment(){
        count.value++;
    }

    onMounted(()=>{
        console.log(count.value)
    })
</script>

<template>
  <button v-on:click="increment">count:{{ count }}</button>
  <br>
  <h1>Got it!</h1>
</template>

    # 综合案例

    利用SrpingBoot后端项目接口完成表格的搜索功能，以 Vue 工程化项目的形式实现

    后端接口：
    'http://localhost:8081/article/getAll'：获取所有文章数据
    'http://localhost:8081/article/search?category=xxx&state=xxx'：根据文章分类和状态返回文章数据

    • 创建并实现Article.vue，包含页面的逻辑和展示
      • axios.get('http://localhost:8081/article/search?category='+searchConditions.value.category+'&state='+searchConditions.value.state)显得冗长且复杂。
      • 展开运算符 ...：{...searchConditions.value} 会将searchConditions对象的属性 { category: 'xxx', state: 'xxx' } 复制到新的对象params中，利用params来传递查询字符串参数。
    <script setup>
    import {onMounted, ref} from 'vue'
    const articleList = ref([])
    const searchConditions = ref({
        category:'', 
        state:''
    })
    import axios from 'axios'

    onMounted(()=>{
        axios.get('http://localhost:8081/article/getAll').then((result)=>{
            articleList.value = result.data
        }).catch((err)=>{
            alert(err)
        })
    })

    const search = function(){
        axios.get('http://localhost:8081/article/search',{params:{...searchConditions.value}}).
        then((result)=>{
            articleList.value = result.data
        }).catch((err)=>{
            alert(err)
        })
    }
</script>

<template>
    <div>
        文章分类: <input type="text" v-model="searchConditions.category">

        发布状态: <input type="text" v-model="searchConditions.state">

        <button v-on:click="search">搜索</button>

        <br />
        <br />
        <table border="1 solid" colspa="0" cellspacing="0">
            <thead>
            <tr>
                <th>文章标题</th>
                <th>分类</th>
                <th>发表时间</th>
                <th>状态</th>
                <th>操作</th>
            </tr>
            </thead>
            <tbody>
            <tr v-for="(article) in articleList" :key="article.title">
                <td>{{article.title}}</td>
                <td>{{article.category}}</td>
                <td>{{article.time}}</td>
                <td>{{article.state}}</td>
                <td>
                    <button>编辑</button>
                    <button>删除</button>
                </td>
            </tr>
            </tbody>
        </table>
    </div>
</template>

    # 函数封装代码优化

    调用后端接口的js代码一般会封装到.js文件中，并且以函数的形式暴露给外部。

    异步操作通常基于 Promise 来执行，它们不会立即返回结果，而是会在未来的某个时刻返回。用async/await来处理异步请求：

    async 用于标记一个函数为异步函数，配合await使用。

    await 会等待函数的完成，然后返回结果。

    • 在/src目录下创建/api/article.js，完成文章获取和搜索函数的封装。

    axios.get()是一个异步操作，因此需要等待后端服务器操作完成后，获取后端返回值，再将这个返回值return。

    import axios from 'axios'

export async function articleGetAllService(){
    return await axios.get('http://localhost:8081/article/getAll').then((result)=>{
        return result.data
    }).catch((err)=>{
        alert(err)
    })
}

export async function articleSearchService(conditions){
    return await axios.get('http://localhost:8081/article/search',{params: conditions}).
    then((result)=>{
        return result.data
    }).catch((err)=>{
        alert(err)
    })
}

    # 请求和响应代码优化

    添加拦截器：在请求或响应被then或catch前拦截并处理。

    • 在/src目录下创建/util/request.js，添加处理后端响应的拦截器
    import axios from 'axios'

const baseURL = 'http://localhost:8081'
const instance = axios.create({baseURL})

//添加响应拦截器
instance.interceptors.response.use(
    result=>{
        return result.data
    },
    err=>{
        alert('服务异常')
        return Promise.reject(err)
    }
)
export default instance
