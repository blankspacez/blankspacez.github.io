    # Vue常用指令

    指令：HTML标签上带有 V-前缀的特殊属性，不同的指令具有不同的含义，可以实现不同的功能。

    常用指令 含义
    v-for 列表渲染，遍历容器的元素或者对象的属性
    v-bind 为HTML标签绑定属性值，如设置 href，css样式等
    v-if/v-else-if/v-else 条件性的渲染某元素，判定为true时渲染，否则不渲染
    v-show 根据条件展示某元素，区别在于切换的是display属性的值
    v-model 在表单元素上创建双向数据绑定
    v-on 为HTML标签绑定事件

    # v-for

    列表渲染，遍历容器的元素或者对象的属性

    v-for = "(item,index)in items"
v-for ="item in items"
    1
    2

    参数说明:

    • items 为遍历的数组
    • item 为遍历出来的元素
    • index 为索引/下标，从0开始; 可以省略

    # v-if / v-else

    基于条件判断，控制创建元素或移除元素（适用于不频繁切换的场景）

    # v-on

    为标签绑定事件

    //function定义在methods中
v-on:event="function"

//简写
@event="function"
    1
    2
    3
    4
    5
    <div id="app">
    <!---绑定函数到单击事件--->
    <button v-on:click="superise">点我有惊喜</button> &nbsp;

            },
            more_superise: function(){
                alert("再点更惊喜")
            } 

        }
    }).mount("#app");//控制html元素
</script>

    # v-model

    在表单上使用，实现双向数据绑定，用于更方便获取和设置表单数据。

    v-model="var"
    1

    以下例子还提供了常见的搜索功能实现案例

    <div id="app">

    <input type="text" v-model="searchCondition"/>

    <button v-on:click="searchData">搜索</button>
    <button v-on:click="clear">清空</button>

    <br />
    <br />
    <table border="1 solid" colspa="0" cellspacing="0">
        <tr>
            <th>文章标题</th>
            <th>分类</th>
            <th>发表时间</th>
            <th>状态</th>
            <th>操作</th>
        </tr>
        <tr v-for="(article,index) in dataList">
            <td>{{article.title}}</td>
            <td>{{article.category}}</td>
            <td>{{article.time}}</td>
            <td>{{article.state}}</td>
            <td>
                <button>编辑</button>
                <button>删除</button>
            </td>
        </tr>
    </table>
</div>
<script type="module">
    //导入vue模块
    import { createApp } from 'https://unpkg.com/vue@3/dist/vue.esm-browser.js'
    //创建vue应用实例
    createApp({
        data() {
            return {
                //定义数据
                articleList: [{
                    title: "医疗反腐绝非砍医护收入",
                    category: "时事",
                    time: "2023-09-5",
                    state: "已发布"
                },
                {
                    title: "中国男篮缘何一败涂地？",
                    category: "篮球",
                    time: "2023-09-5",
                    state: "草稿"
                },
                {
                    title: "华山景区已受大风影响阵风达7-8级，未来24小时将持续",
                    category: "旅游",
                    time: "2023-09-5",
                    state: "已发布"
                }],

                searchCondition:"",
                dataList:[]
            }
        },
        created() {
            // 初始化时显示所有数据
            this.dataList = [...this.articleList]
        },
        methods:{
            clear:function(){
                this.searchCondition=""
            },
            searchData:function(){
                if(this.searchCondition.trim()===""){
                    this.dataList = [...this.articleList]
                }else{
                    this.dataList = this.articleList.filter(article => {
                        return (
                            article.title.includes(this.searchCondition) ||
                            article.category.includes(this.searchCondition) ||
                            article.time.includes(this.searchCondition) ||
                            article.state.includes(this.searchCondition) 
                        )
                    })
                }
            }
        }

    }).mount("#app")//控制html元素
</script>

    # Vue的生命周期

    生命周期的八个阶段：每个阶段会自动执行一个生命周期方法(钩子)，让开发者有机会在特定的阶段执行自己的代码。

    常用mounted()函数：在页面加载完成时，发送异步请求，加载数据，渲染页面。

    状态 生命周期
    beforeCreate 创建前
    created 创建后
    beforeMount 载入前
    mounted 挂载完成
    beforeUpdate 数据更新前
    updated 数据更新后
    beforeUnmount 组件销毁前
    unmounted 组件销毁后

    # 综合案例

    呈现效果：利用SrpingBoot后端项目接口完成表格的搜索功能

    后端接口：
    'http://localhost:8081/article/getAll'：获取所有文章数据
    'http://localhost:8081/article/search?category=xxx&state=xxx'：根据文章分类和状态返回文章数据

    技术要点：

    • 钩子函数mounted中，获取所有的文章数据
    • 使用v-for指令,把数据渲染到表格上展示
    • 使用v-model指令完成表单数据的双向绑定
    • 使用v-on指令为搜索按钮绑定单击事件

    🐴🐴🐴🐴🐴🐴🐴🐴🐴🐴🐴🐴

    <div id="app">

    文章分类: <input type="text" v-model="searchConditions.category">

    发布状态: <input type="text" v-model="searchConditions.state">

    <button v-on:click="search">搜索</button>

    <br />
    <br />
    <table border="1 solid" colspa="0" cellspacing="0">
        <tr>
            <th>文章标题</th>
            <th>分类</th>
            <th>发表时间</th>
            <th>状态</th>
            <th>操作</th>
        </tr>
        <tr v-for="(article) in articleList">
            <td>{{article.title}}</td>
            <td>{{article.category}}</td>
            <td>{{article.time}}</td>
            <td>{{article.state}}</td>
            <td>
                <button>编辑</button>
                <button>删除</button>
            </td>
        </tr>
    </table>
</div>
<script src="https://unpkg.com/axios/dist/axios.min.js"></script>
<script type="module">
    //导入vue模块
    import { createApp} from 'https://unpkg.com/vue@3/dist/vue.esm-browser.js'

    //创建vue应用实例
    createApp({
        data() {
            return {
                articleList:[],
                searchConditions:{
                    category:'',
                    state:''
                }
            }
        },
        methods:{
            search: function(){
                axios.get('http://localhost:8081/article/search?category='+this.searchConditions.category+'&state='+this.searchConditions.state).
                then((result)=>{
                    this.articleList = result.data
                }).catch((err)=>{
                    alert(err)
                })
            }
        },
        mounted(){
            axios.get('http://localhost:8081/article/getAll').then((result)=>{
                this.articleList = result.data
            }).catch((err)=>{
                alert(err)
            });
        }
    }).mount("#app");//控制html元素

</script>
