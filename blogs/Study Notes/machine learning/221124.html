    摘要

    机器学习实训课程笔记 😈 😈 😈

    开发工具：annaconda + jupyter notebook

    # 时序特征处理

    在进行数据分析或量化研究时，总避免不了时间序列数据的处理，时间序列是指在一定时间内按时间顺序测量的某个变量的取值序列。常见的时间序列数据有一天内随着时间变化的温度序列，又或者交易时间内不断波动的股票价格序列等等。

    pandas提供了各种时序特征处理的函数接口。

    参考文章
    👉Pandas数据处理——玩转时间序列数据 (opens new window)
    👉时间序列数据的特征工程总结 (opens new window)

    关于特征处理，可以注册一个Kaggle进行学习。

    # tsfresh

    tsfresh 用于从时间序列或其他序列数据中进行系统特征工程。

    tsfresh官网 (opens new window)

    利用tsfresh来提取时序特征十分方便，只需要调用库函数即可，如果利用pandas来一步步手动计算、提取特征，那么过程会非常繁琐。

    from tsfresh.examples.robot_execution_failures import download_robot_execution_failures,load_robot_execution_failures

# 加载数据集
timeseries,y = load_robot_execution_failures() #timeseries为DataFrame类型的Data，y为Target


# 某一列的特征可视化
import matplotlib.pyplot as plt

timeseries[timeseries['id'] == 3].plot(subplots = True,sharex = True,figsize = (10,10))
plt.show()

timeseries[timeseries['id'] == 88].plot(subplots = True,sharex = True,figsize = (10,10))
plt.show()

# 提取特征
from tsfresh import extract_features

extract_features = extract_features(timeseries,column_id = 'id',column_sort='time')
extract_features.head()

from tsfresh import select_features
extract_features.fillna(0,inplace=True)

X_selected = select_features(extract_features, y)

# 机器学习
from sklearn.ensemble import RandomForestClassifier
from sklearn.model_selection import train_test_split
from sklearn import metrics
#训练
X_train,X_test,y_train,y_test = train_test_split(X_selected,y,test_size=0.3)
rf = RandomForestClassifier(n_estimators=20,max_depth=5)
rf.fit(X_train,y_train)
#查看模型预测效果
pred = rf.predict_proba(X_test)[:,1]
fpr,tpr,thres = metrics.roc_curve(y_test,pred)
metrics.auc(fpr,tpr)
