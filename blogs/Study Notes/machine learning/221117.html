    # 特征工程(feature engineering)

    • 特征即数据中抽取出来的对结果预测有用的信息

    • 特征工程是使用专业背景知识和技巧处理数据，使得特征能在机器学习算法上发挥更好的作用的过程。

    # 特征工程的意义

    • 更好的特征意味着更强的灵活度

    • 更好的特征意味着只需用简单模型

    • 更好的特征意味着更好的结果

    # 二值化

    指定阈值为n，将大于阈值的数据转化成1，小于等于阈值的数据转化成0。

    from sklearn.preprocessing import Binarizer
sample=X[0:20,0] # 对X的第一列的0-20行数据二值化

model=Binarizer(threshold=6)  #二值化的阈值为6，即大于6的为1，小于6的为0
result=model.fit_transform(sample.reshape(-1,1)) # 参数为行向量

    # 分箱

    分箱的主要目的是去噪，将连续数据离散化。

    • 等频分箱：每个区间内包括的值一样多
    • 等距分箱：每两区间之间的距离是一样的
    from sklearn.preprocessing import KBinsDiscretizer

train_set=X[10:,0]
test_set=X[0:10,0]

model=KBinsDiscretizer(n_bins=5,encode='ordinal',strategy='uniform') # 等距分箱
'''
model=KBinsDiscretizer(n_bins=5,encode='ordinal',strategy='quantile')# 等频分箱
'''

model.fit(train_set.reshape(-1,1))
train_trans=model.transform(train_set.reshape(-1,1)).reshape(-1)
test_trans=model.transform(test_set.reshape(-1,1)).reshape(-1)

bin_edge = model.bin_edges_[0]  #查看箱子的边缘

# 画图直观地观察箱子的边缘
import seaborn as sns
fig, ax = plt.subplots()
sns.distplot(train_set, hist=True, kde=True)
for edge in bin_edge :
    line = ax.axvline(edge, color='r')
    
ax.set_title("分布")
ax.set_xlabel("变量数值")
ax.set_ylabel("频率")

    # 缩放

    • Z标准化缩放相当于对数据做正态分布的标准化：

    value=x−μσvalue=\frac{x-\mu}{\sigma} value=​σ​​x−μ​​

    • 最大最小值缩放即将区间[a,b]上的值缩放到[0,1]上
      指定最小值为a，最大值为b，则

    value=x−ab−avalue=\frac{x-a}{b-a} value=​b−a​​x−a​​

    • 鲁棒优化对于数据中心化和数据的缩放健壮性有着更强的参数调节能力，常使用于对异常值比较敏感的情况。
      指定两个分位数a和b，则

    value=x−medianPb−Pavalue=\frac{x-median}{P_b-P_a} value=​P​b​​−P​a​​​​x−median​​

    '''
from sklearn.preprocessing import StandardScaler #标准缩放
from sklearn.preprocessing import MinMaxScaler #最大最小值缩放
'''
from sklearn.preprocessing import RobustScaler #鲁棒缩放
# 画图比较缩放前后的数据分布
fig,ax = plt.subplots(2,1,figsize = (16,9))
sns.distplot(X[:,0], hist=True, kde=True,ax = ax[0])

ax[0].set_title("分布")
ax[0].set_xlabel("变量数值")
ax[0].set_ylabel("频率")

'''
model = StandardScaler()
model = MinMaxScaler(feature_range=(0,10)) #参数指定最大最小值
'''
model = RobustScaler(quantile_range=(25.0,75.0)) #参数指定两个分位数

model.fit(X[:,0].reshape(-1,1))
result = model.transform(X[:, 0].reshape(-1,1)).reshape(-1)

sns.distplot(result, hist=True,kde=True, ax = ax[1])

ax[1].set_title("分布")
ax[1].set_xlabel("变量数值")
ax[1].set_ylabel("频率")

    用模型预测的方式填充缺省：

    from sklearn.impute import IterativeImputer
from sklearn.tree import DecisionTreeRegressor

train_set = X[10:,:]
test_set = X[0: 10,:]
# 设置缺省
test_set[4,2] = np.nan
test_set[5,0] = np.nan
test_set[7,1] = np.nan
train_set[3,0] = np.nan
train_set[6,0] = np.nan
train_set[10,2] = np.nan

impute_es = DecisionTreeRegressor() #决策树模型

imputer = IterativeImputer(estimator=impute_es)
imputer.fit(train_set)

imputr_train = imputer.transform(train_set)
imputr_test = imputer.transform(test_set)

# 查看填充前后的数据对比
import pandas as pd
pd.DataFrame(test_set)
pd.DataFrame(imputr_test)
