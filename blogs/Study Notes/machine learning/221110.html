    # 随机森林（RandomForest）

    # bagging思想

    • Bagging是bootstrap aggregating的缩写，使用了bootstrap思想。
    • Bagging降低过拟合风险，提高泛化能力。

    输入为样本集D=(x1,y1),(x2,y2),⋯,(xm,ym)D={(x_1,y_1),(x_2,y_2),\cdots,(x_m,y_m)}D=(x​1​​,y​1​​),(x​2​​,y​2​​),⋯,(x​m​​,y​m​​)

    • 对训练集进行T次随机采样，每次采集m个样本，得到包含m个样本的采样集DmD_mD​m​​；
      用采样集DmD_mD​m​​训练基学习器Gm(x)G_m(x)G​m​​(x)

    • 分类场景：则T个学习器投出最多票数的类别为最终类别。
      回归场景：则T个学习器得到的回归结果进行算术平均得到的值为最终的模型输出。

    # 随机森林算法

    • RF（随机森林）是一种基于树模型的Bagging的优化版本。核心思想依旧是bagging，但是做了一些独特的改进。
    • RF使用了CART决策树作为基学习器。

    输入为样本集D=(x1,y1),(x2,y2),⋯,(xm,ym)D={(x_1,y_1),(x_2,y_2),\cdots,(x_m,y_m)}D=(x​1​​,y​1​​),(x​2​​,y​2​​),⋯,(x​m​​,y​m​​)

    • 对训练集进行T次随机采样，每次采集m个样本，得到包含m个样本的采样集DmD_mD​m​​；
      用采样集DmD_mD​m​​训练基学习器Gm(x)G_m(x)G​m​​(x)。在训练决策树模型的结点的时候，在结点上所有的样本特征中选择一部分样本特征，在这些随机选择的部分样本特征中选择一个最优的特征来做决策树的左右子树划分。（每个学习器着重学习的特征不同）

    • 分类场景：则T个基模型(决策树)投出最多票数的类别为最终类别。
      回归场景：则T个基模型(回归树)得到的回归结果进行算术平均得到的值为最终的模型输出。

    # 代码实战

    独热编码（One-Hot Encoding）

    又称一位有效编码，其方法是使用N位状态寄存器来对N个状态进行编码，每个状态都有它独立的寄存器位，并且在任意时候，其中只有一位有效。即，只有一位是1，其余都是零值。

    例子：
    地区特征：["北京"，"上海，"深圳"]（这里N=3，三维数据）：
    北京 => 100
    上海 => 010
    深圳 => 001
    工作特征：["演员"，"厨师"，"公务员"，"工程师"，"律师"]（这里N=5，五维数据）：
    演员 => 10000
    老师 => 01000
    公务员 => 00100
    工程师 => 00010
    消防员 => 00001

    # AdaBoost

    # boost思想

    • Boosting算法和Bagging算法一样,也是一种集成学习算法。

    • Boosting算法不是对样本进行随机抽样构造训练集，而是重点关注被前面的弱分类器错分的样本。每一次迭代都根据上一次迭代的预测结果对样本进行加权，所以随着迭代不断进行，误差会越来越小。

    弱学习器常指泛化性能略优于随即猜测的学习器（例如在二分类问题上精度略高于50%的分类器）。单个弱分类器的性能一般，仅比随即猜测强，但将多个弱分类器组合在一起，能构造出一个非常准确的强分类器。弱分类器的名字因此而来。

    # AdaBoost算法

    AdaBoost算法的全称是自适应提升（Adaptive Boost)，由Freund等提出，是一种典型的Boosting方法。

    AdaBoost算法是一种用于分类问题的算法，它用弱分类器的线性组合来构造强分类器。AdaBoost也可以解决回归问题。

    训练流程：

    • 依次训练每一个弱分类器，并得到它们的权重值。
    • 训练样本带有权重值，其中初始时所有样本的权重值相等。
    • 在训练过程中，被前面的弱分类器错分的样本会加大权重，反之会减小权重。这样接下来的弱分类器会更加关注这些难分的样本。
    • 弱分类器的权重值根据它的准确率构造，精度越高的弱分类器权重越大。
    比较项目 随机森林 AdaBoost
    决策树规模 大 小
    是否对样本随机采样 是 否
    是否对特征随机采样 是 否
    弱分类器是否有权重 无 有
    训练样本是否有权重 无 有
    是否支持多分类 是 不直接支持
    是否直持回归 是 不直接支持

    # 支持向量机（Support Vector Machine）

    # SVM介绍

    • SVM属于有监督学习模型，在机器学习、计算机视觉、数据挖掘中广泛应用，主要用于解决数据分类问题。还可以用于回归问题。

    • 通常SVM用于二元分类问题，对于多元分类可将其分解为多个二元分类问题，再进行分类。

    # SVM基本原理

    分类问题的关键在于找到一条决策边界

    • 线性分类问题：决策边界容易寻找
    • 非线性问题：通过升维手段寻找

    SVM算法的核心理念是计算一条最为中立的分离直线。该直线既不偏向训练数据中的正采样，也不偏向负采样。

    SVM算法试图分清边界上的样本点，从而不断寻找最优超平面。
