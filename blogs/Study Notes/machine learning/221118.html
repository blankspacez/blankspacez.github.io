    # TF-IDF算法

    Ref: TF-IDF算法介绍及实现 (opens new window) & 特征缩放的效果：从词袋到_TF-IDF (opens new window)

    TF-IDF（term frequency - inverse document frequency，词频-逆向文件频率）是一种用于信息检索与文本挖掘的常用加权技术。

    TF-IDF的主要思想是：如果某个单词在一篇文章中出现的频率TF高，并且在其他文章中很少出现，则认为该词或者短语具有很好的类别区分能力，适合用来分类。

    # TF(term frequency，词频)

    词频（TF）表示词条（关键字）在文本中出现的频率。

    # IDF(inverse document frequency，逆向文件频率)

    逆向文件频率（IDF）的含义：如果包含词条w的文档越少, 则IDF越大，则说明该词条w具有很好的区分能力。

    # TF-IDF

    • TF-IDF实际上是TF*IDF

    • 某一特定文件内的高词语频率，以及该词语在整个文件集合中的低文件频率，根据TF-IDF算法具有很高的权重，从而说明这样的词语更为“关键”。

    import jieba.analyse

jieba.analyse.extract_tags(sentence, topK=20, withWeight=False, allowPOS=())

sentence 为待提取的文本

    # 🐴🐴🐴来

    import jieba
import docx
import jieba.analyse
import os
import math

dirName="D:/深度学习/机器学习实训/TF-IDF作业"
filelist = os.listdir(dirName)

idf_dic={}
doc_count=len(filelist)

# 构造自定义IDF语料库
for fileName in filelist:
    if(fileName.split('.')[1]=="docx"):
        docStr = docx.Document(dirName+'/'+fileName)
        text=""
        for paragraph in docStr.paragraphs:
            text += paragraph.text
        words = jieba.lcut(text)
        for word in words:
            if(word != ' '):
                 idf_dic[word] = idf_dic.get(word, 0.0) + 1.0
for k,v in idf_dic.items():
    idf = '%.10f' % (math.log(doc_count / (1.0 + v))) # 公式计算IDF，小数点后保留10位
    idf_dic[k] = idf
            
with open('D:/深度学习/机器学习实训/my_idf.txt','w',encoding='utf-8') as f:
    for k in idf_dic:
        if k != '\n':
            f.write(k + ' ' + idf_dic[k] + '\n') #写入txt文件，注意utf-8，否则jieba不认


# 设置自定义IDF语料库和停用词表
jieba.analyse.set_idf_path("D:/深度学习/机器学习实训/my_idf.txt")  
jieba.analyse.set_stop_words("D:/深度学习/机器学习实训/cn_stopwords.txt") 

# 提取关键词
for fileName in filelist:
    if(fileName.split('.')[1]=="docx"):
        docStr = docx.Document(dirName+'/'+fileName)
        parStr=""
        for paragraph in docStr.paragraphs:
            parStr += paragraph.text
        keywords=jieba.analyse.extract_tags(parStr, topK=10, withWeight=False, allowPOS=())
        print(fileName,keywords)
