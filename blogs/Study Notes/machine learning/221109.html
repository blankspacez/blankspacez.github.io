    # 预备知识

    # 过拟合（overfitting）

    过拟合其实就是为了得到一致假设而使得假设过于地严格。使得其在训练集上的表现非常地完美，但是在训练集以外的数据集却表现不好。

    # 正则化（regularization）

    • L1正则化（Lasso正则化）
      L1正则化项：λn∑∣ωj∣\frac{\lambda}{n}\sum|\omega_{j}|​n​​λ​​∑∣ω​j​​∣
      特点：L1会趋向于产生少量的特征，而其他的特征都是0。因为最优的参数值很大概率出现在坐标轴上，这样就会导致某一维的权重为0 ，产生稀疏权重矩阵。

    • L2正则化（Ridge正则化）
      L1正则化项：λ2n∑ωj2\frac{\lambda}{2n}\sum\omega_{j}^2​2n​​λ​​∑ω​j​2​​
      特点：L2会选择更多的特征，这些特征都会接近于0。最优的参数值很小概率出现在坐标轴上，因此每一维的参数都不会是0，但会使每一项趋近于0。

    # 决策树🌳

    决策树（decision tree）是一种基本的分类与回归方法。

    • 学习过程：通过分析训练样本来划分内部结点对应的属性。

    • 预测过程：将测试样例从根结点开始，沿着划分的属性所构成的“判定测试序列”下行，直到叶子结点。

    # ID3

    • ID3选择信息增益最大的属性作为决策树的根结点，主要用于分类。

    • 问题：对取值数目较多的属性有偏好。

    # 信息熵（entropy）

    若当前样本集合D中，第k类样本（总共|y|类）所占的比例为pkp_kp​k​​，则D的信息熵定义为：

    Ent(D)=−∑k=1∣y∣pklog2pk Ent(D)=-\sum_{k=1}^{|y|}p_k\log_2p_k Ent(D)=−​k=1​∑​∣y∣​​p​k​​log​2​​p​k​​

    Ent(D)的值越小，则D的纯度越高。

    # 信息增益（information gain）

    设离散属性a的取值有{a1,a2,a3,⋯,ana^1,a^2,a^3,\cdots,a^na​1​​,a​2​​,a​3​​,⋯,a​n​​}，且对应的样本的集合为{D1,D2,D3,⋯,DvD^1,D^2,D^3,\cdots,D^vD​1​​,D​2​​,D​3​​,⋯,D​v​​}，则以属性aaa对集合进行划分的信息增益为：

    Gain(D,a)=Ent(D)−∑v=1V∣Dv∣∣D∣Ent(Dv) Gain(D,a)=Ent(D)-\sum_{v=1}^V\frac{|D^v|}{|D|}Ent(D^v) Gain(D,a)=Ent(D)−​v=1​∑​V​​​∣D∣​​∣D​v​​∣​​Ent(D​v​​)

    # C4.5

    先从候选属性集合中选取信息增益高于平均水平的，再从中选取信息增益率最高的。主要用于分类。

    # 信息增益率（information gain ratio）

    Gainratio(D,a)=Gain(D,a)Iv(a) Gain_ratio(D,a)=\frac{Gain(D,a)}{Iv(a)} Gain​r​​atio(D,a)=​Iv(a)​​Gain(D,a)​​

    # CART

    • CART从候选属性集合中选取基尼指数最小的，用于分类和回归

    • 目的：最小化不纯度，而不是最大化信息增益

    # 决策树剪枝操作

    剪枝：为了防止分支过多可能引发的过拟合，从而主动去掉一些分支来降低过拟合的风险。

    预剪枝 后剪枝
    基本策略 提前终止某些分支的生长 生成一棵完整的树之后再进行剪枝(自底向上)
    时间开销 训练时间开销降低，测试时间开销降低 训练时间开销增加，测试时间开销降低
    风险 过拟合风险降低,欠拟合风险增加 过拟合风险降低，欠拟合风险基本不变

    • 使用留出法对剪枝前后的决策树进行评估，若划分该属性后验证集的精度下降，则进行剪枝。

    • 泛化性能:后剪枝通常优于预剪枝

    # “特殊值”属性的处理

    # 连续值属性的处理

    • 举例：西瓜🍉的含糖率和密度等属性都是一系列连续的值

    • 处理方式：连续数据离散化，将数据用二分法进行划分（留出法比较精度）

    # 缺省值属性的处理

    基本思路：样本赋权，权重划分

    举例：计算信息增益率时，仅考虑无缺省值的数据，但会在比较信息增益率大小的时候为其赋上一定的权重

    # 回归树（regression tree）

    # 回归树模型

    • 把整个特征空间切分成不相交的子区域

    • 每个区域预估成该区域样本的均值

    经典案例
    根据从业年限和表现，来预估棒球运动员的工资。
    横坐标表示年限，纵坐标表示表现。红黄表示高工资，蓝绿表示低工资。

    # 回归树构建方法

    回归树构建步骤：

    • 把整个特征空间 X 切分成 J 个没有重叠的区域R1,R2,R3,⋯,RJR_1,R_2,R_3,\cdots,R_JR​1​​,R​2​​,R​3​​,⋯,R​J​​
    • 其中区域RjR_jR​j​​中的每个样本我们都给一样的预测结果yRj~=1n∑j∈Rjyj\tilde{y_{R_j}}=\frac{1}{n}\sum_{j\in{R_j}}y_j​y​R​j​​​​​~​​=​n​​1​​∑​j∈R​j​​​​y​j​​，其中n是RjR_jR​j​​中的总样本数。

    即找到使RSS最小的划分R1,R2,R3,⋯,RJR_1,R_2,R_3,\cdots,R_JR​1​​,R​2​​,R​3​​,⋯,R​J​​

    RSS=∑j=1J∑i∈Rj(yi−yRj~)2 RSS=\sum_{j=1}^J\sum_{i\in{R_j}}(y_i-\tilde{y_{R_j}})^2 RSS=​j=1​∑​J​​​i∈R​j​​​∑​​(y​i​​−​y​R​j​​​​​~​​)​2​​

    采用自顶向下的贪婪式递归方案

    • 自顶向下：从所有样本开始，不断从当前位置，把样本切分到2个分支里。
    • 贪婪：每一次的划分，只考虑当前最优，而不回过头考虑之前的划分。

    回归树剪枝

    • 通过添加正则化项防止过拟合
    • 通过交叉验证法选择一个α\alphaα，剪枝后的🌳应当使下列式子的值最小，其中|T|为回归树叶子结点的个数。

    ∑j=1∣T∣∑i∈Rj(yi−yRj~)2+α∣T∣ \sum_{j=1}^{|T|}\sum_{i\in{R_j}}(y_i-\tilde{y_{R_j}})^2+\alpha|T| ​j=1​∑​∣T∣​​​i∈R​j​​​∑​​(y​i​​−​y​R​j​​​​​~​​)​2​​+α∣T∣
