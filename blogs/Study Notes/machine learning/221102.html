    # 回归（Regression）算法指标

    # 残差（实际值与预测值差值）

    residual(y,y^)=1m∑i=1m(yi−yi^) residual(y,\hat{y})=\frac{1}{m}\sum_{i=1}^{m}(y_i-\hat{y_i}) residual(y,​y​^​​)=​m​​1​​​i=1​∑​m​​(y​i​​−​y​i​​​^​​)

    当实际值分布在拟合曲线两侧时，对于不同样本而言yi−yi^y_i-\hat{y_i}y​i​​−​y​i​​​^​​有正有负，相互抵消，因此我们想到采用预测值和真实值之间的距离来衡量。

    # 平均绝对误差MAE

    MAE(y,y^)=1m∑i=1m∣yi−yi^∣ MAE(y,\hat{y})=\frac{1}{m}\sum_{i=1}^{m}|y_i-\hat{y_i}| MAE(y,​y​^​​)=​m​​1​​​i=1​∑​m​​∣y​i​​−​y​i​​​^​​∣

    MAE虽能较好衡量回归模型的好坏，但是绝对值的存在导致函数不光滑，在某些点上不能求导，可以考虑将绝对值改为残差的平方，这就是均方误差。

    # 均方误差 MSE & 均方根误差 RMSE

    MSE(y,y^)=1m∑i=1m(yi−yi^)2 MSE(y,\hat{y})=\frac{1}{m}\sum_{i=1}^{m}(y_i-\hat{y_i})^2 MSE(y,​y​^​​)=​m​​1​​​i=1​∑​m​​(y​i​​−​y​i​​​^​​)​2​​

    RMSE(y,y^)=1m∑i=1m(yi−yi^)2 RMSE(y,\hat{y})=\sqrt{\frac{1}{m}\sum_{i=1}^{m}(y_i-\hat{y_i})^2} RMSE(y,​y​^​​)=​⎷​​​​​​​m​​1​​​i=1​∑​m​​(y​i​​−​y​i​​​^​​)​2​​​​​

    # 决定系数 R2R^2R​2​​

    最为常用

    R2=1−1m∑i=1m(yi−yi^)21m∑i=1m(yi−yi¯)2=1−MSE(y,y^)Var(y) R^2=1-\frac{\frac{1}{m}\sum_{i=1}^{m}(y_i-\hat{y_i})^2}{\frac{1}{m}\sum_{i=1}^{m}(y_i-\bar{y_i})^2}=1-\frac{MSE(y,\hat{y})}{Var(y)} R​2​​=1−​​m​​1​​∑​i=1​m​​(y​i​​−​y​i​​​¯​​)​2​​​​​m​​1​​∑​i=1​m​​(y​i​​−​y​i​​​^​​)​2​​​​=1−​Var(y)​​MSE(y,​y​^​​)​​

    如果结果是0，就说明模型预测不能预测因变量。 如果结果是1。就说明是函数关系。 如果结果是0-1之间的数，就是我们模型的好坏程度(越大越好)。

    # 分类（Classification）算法指标

    # 准确率（查准率） Precision和召回率（查全率）Recall

    指标 说明 应用场景举例
    Precision Precison=TPFP+TPPrecison=\frac{TP}{FP+TP}Precison=​FP+TP​​TP​​，分类器预测的正样本中预测正确的比例 嫌疑人定罪
    Recall Recall=TPTP+FNRecall=\frac{TP}{TP+FN}Recall=​TP+FN​​TP​​，分类器所识别出的正样本占所有正样本的比例 地震的预测

    # ROC和AUC

    • ROC曲线由两个变量TPR和FPR组成

    • AUC不依赖于阈值。AUC值为ROC曲线所覆盖的区域面积，显然，AUC越大，分类器分类效果越好。
      AUC = 1，是完美分类器。绝大多数预测的场合，不存在完美分类器。
      0.5 < AUC < 1，优于随机猜测。这个分类器（模型）妥善设定阈值的话，能有预测价值。
      AUC = 0.5，跟随机猜测一样（例：丢铜板），模型没有预测价值。
      AUC < 0.5，比随机猜测还差；但只要总是反预测而行，就优于随机猜测。

    AUC计算方法:

    • AUC为ROC曲线下的面积，那直接计算面积可得。
    • 根据AUC的物理意义（AUC只关注正负样本之间的排序），计算正样本预测结果大于负样本预测结果的概率。
    • 首先把所有样本按照score排序，依次用rank表示他们，如最大score的样本，rank=n （n=n0+n1n=n_0+n_1n=n​0​​+n​1​​，其中n0n_0n​0​​为负样本个数，n1n_1n​1​​为正样本个数），其次为n-1。
    AUC=\frac{\sum_{正样本}{rank(score)}-\frac{n_1\times(n_1+1)}{2}}{n_0\times n_1}

    # 逻辑函数

    逻辑回归是数学家为了得到[0.1]之间输出，而使用线性回归作为逻辑函数的输入，所创造出来的东西。

    来看看一个简单的逻辑函数——Sigmoid函数：

    • 线性回归的一般表达式为y=b+ω1x1+ω2x2+⋯+ωnxny=b+\omega_1x_1+\omega_2x_2+\cdots+\omega_nx_ny=b+ω​1​​x​1​​+ω​2​​x​2​​+⋯+ω​n​​x​n​​
      可以写成y=ωTxy=\omega^{T}xy=ω​T​​x（w0=b,x0=1w_0=b,x_0=1w​0​​=b,x​0​​=1）的形式
    • 将线性回归作为逻辑回归的输入，则输出逻辑回归P=11+e−z(z=ωTx)P=\frac{1}{1+e^{-z}}(z=\omega^{T}x)P=​1+e​−z​​​​1​​(z=ω​T​​x)

    # 最大似然估计

    当模型已定，参数未知的情况，最大似然估计可以帮助我们求解出参数。

    比如有一堆白球和黑球，每次取出一个球然后放回，重复100次，则取出70次黑球和30次白球的概率为:P=p70(1−p)30P=p^{70}(1-p)^{30}P=p​70​​(1−p)​30​​。现要使该概率最大，则可以求出白球黑球的比为3：7。

    将该场景类比我们的二分类问题：

    将单样本看成一个事件，则该事件发生的概率为P(yi∣xi)=pyi(1−p)1−yiP(y_i|x_i)=p^{y_i}(1-p)^{1-y_i}P(y​i​​∣x​i​​)=p​y​i​​​​(1−p)​1−y​i​​​​，因为是二分类问题，因此yiy_iy​i​​=0或1。

    考虑所有的样本，可以得出公式：

    P=P(y1∣x1)P(y2∣x2)⋯P(yn∣xn)=∏n=1Npyi(1−p)1−yi P=P(y_1|x_1)P(y_2|x_2) \cdots P(y_n|x_n)=\prod_{n=1}^Np^{y_i}(1-p)^{1-y_i} P=P(y​1​​∣x​1​​)P(y​2​​∣x​2​​)⋯P(y​n​​∣x​n​​)=​n=1​∏​N​​p​y​i​​​​(1−p)​1−y​i​​​​
