    # Pandas 数据结构 - Series

    Pandas Series 类似表格中的一个列（column），类似于一维数组，可以保存任何数据类型。

    在操作Series的时候，基本上可以把Series看成NumPy中的ndarray数组来进行操作，ndarray数组的绝大多数操作都可以应用到Series上

    Series 由索引（index）和列组成，函数如下：

    pandas.Series( data, index, dtype, name, copy)
    1

    参数

    • data：可以是 python 对象、numpy 的 ndarray 、一个标量（定值，如 8)
    • index：数据索引标签，如果不指定，默认从 0 开始。
    • dtype：数据类型，默认会自己判断。
    • name：设置名称。
    • copy：拷贝数据，默认为 False。

    # 切片

    • 类似NumPy的切片操作

    示例

    import pandas as pd

obj = pd.Series([4,7,-5,3], index=['d','f','b','c'])
print(obj[1:3])
print(obj['b':'c'])
print(obj[1::2])  # 从i=1开始到结尾（包含），步长为2

    # 运算

    • NumPy中数组运算，在Series中都可以使用，并且Series进行数组运算的时候，索引与值之间的映射关系不会发生改变

    示例

obj1 = pd.Series([4,7,-5,3], index=['d','f','b','c'])
# 更改索引
obj1.index = ['one','two','three','four']
obj2 = pd.Series([195, 73,1], index=['a','b','one'])
print(obj1 + obj2)
'''处理缺省时，将int64转换为了float64

two      NaN
dtype: float64
'''

obj1.dtype # 数据类型
obj1.array # 返回值的数列

    # isnull() 函数

    • 缺失值检测

    语法

    pd.isnull(obj)
obj.isnull()
    1
    2

    示例

    import numpy as np
import pandas as pd


obj1 = pd.Series([4,7,-5,3], index=['d','f','b','c'])
obj1.index = ['one','two','three','four']
obj2 = pd.Series([195, 73,1], index=['a','b','one'])
res = obj1 + obj2
mask = ~res.isnull()
''' maskr仍为Series结构：
a        False
b        False
four     False
one       True
three    False
two      False
dtype: bool
'''
print(res[mask])
'''
one    5.0
dtype: float64
'''

    # 其他函数

    s = pd.Series([1,2,3,4], name='数字')
s.add(1) # 每个元素加1 abs()
s.add_prefix(3) # 给索引前加个3，升位
s.add_suffix(4) # 同上，在后增加
s.sum() # 总和
s.count() # 数量，长度
s.agg('std') # 聚合，仅返回标准差, 与 s.std() 相同
s.agg(['min', 'max']) # 聚合，返回最大最小值
s.align(s2) # 联接
s.any() # 是否有为假的
s.all() # 是否全是真
s.append(s2) # 追加另外一个 Series
s.apply(lambda x:x+1) # 应用方法
s.empty # 是否为空
s3 = s.copy() # 深拷贝
