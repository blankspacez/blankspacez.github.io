    # 1 部署Java项目

    任务目标：将本地名为的xxx-demoSpringBoot项目部署到服务器的Docker中。

    # 1.1 准备工作

    # 1.1.1 打包SpringBoot项目

    打开IDEA，找到侧边栏的Maven，点击Lifecycle，选择package，打包项目。

    打包完成后，会在target目录下生成xxx-demo.jar。

    # 1.1.2 编写Dockerfile文件

    # 基础镜像
FROM openjdk:17-jre-buster
# 设定时区
ENV TZ=Asia/Shanghai
RUN ln -snf /usr/share/zoneinfo/$TZ /etc/localtime && echo $TZ > /etc/timezone
# 拷贝jar包
COPY xxx-demo.jar /app.jar
# 入口
ENTRYPOINT ["java", "-jar", "/app.jar"]

    # 1.1.3 准备mysql数据库

    创建自定义的my-network网络：

    docker network create my-network
    1

    通过Docker Hub安装并运行mysql镜像，并连接到my-network网络：

    docker run -d --name mysql -p 3306:3306 --network my-network -e MYSQL_ROOT_PASSWORD=123456 mysql:latest
    1

    Shell会话窗口连接mysql, 创建数据库和表。

    # 1.2 开始部署

    # 1.2.1 上传文件

    将打包的xxx-demo.jar和Dockerfile文件上传到服务器。

    # 1.2.2 构建镜像

    进入Dockerfile文件所在目录，执行以下命令：

    docker build -t xxx-demo .
    1

    查看是否构建成功：

    docker images
    1

    确保端口未被占用之后，运行镜像（和mysql连接到同一个自定义网络）：

    docker run -d -p 8080:8080 --name xxx-demo --network my-network xxx-demo
    1

    OVER 👊 👊 👊

    # 2 部署静态资源nginx

    • 准备工作

    准备配置文件nginx.conf和静态资源存放目录html，将nginx.conf和html文件上传到服务器。

    • 开始部署

    通过Docker Hub安装并运行nginx镜像，完成目录的挂载，并连接到my-network网络：

    docker run -d \
    --name nginx \
    -p 18080:18080 \
    -p 18081:18081 \
    --network my-network \
    -v /root/nginx/nginx.conf:/etc/nginx/nginx.conf \
    -v /root/nginx/html:/usr/share/nginx/html \
    nginx:latest

    # 3 DockerCompose快速部署

    DockerCompose是Docker官方提供的用于管理多个Docker容器的工具。DockerCompose的yaml文件定义了多个容器的配置，包括镜像、端口映射、挂载目录等。

    # 3.1 编写yaml文件

    DockerCompose的语法非常简单，只需要在yaml文件中定义好容器的配置即可，可以参考docker run命令的编写。

    将相关文件上传至服务器，即可通过DockerCompose部署上面的Java项目和静态资源nginx：

    version: "3.8"

services:
  mysql:
    image: mysql
    container_name: mysql
    ports:
      - "3306:3306"
    environment:
      TZ: Asia/Shanghai
      MYSQL_ROOT_PASSWORD: 123456
    volumes:
      - "./mysql/conf:/etc/mysql/conf.d"
      - "./mysql/data:/var/lib/mysql"
      - "./mysql/init:/docker-entrypoint-initdb.d"
    networks:
      - my-network
  xxx-demo:
    build: 
      context: .
      dockerfile: Dockerfile
    container_name: xxx-demo
    ports:
      - "8080:8080"
    networks:
      - my-network
    depends_on:
      - mysql
  nginx:
    image: nginx
    container_name: nginx
    ports:
      - "18080:18080"
      - "18081:18081"
    volumes:
      - "./nginx/nginx.conf:/etc/nginx/nginx.conf"
      - "./nginx/html:/usr/share/nginx/html"
    depends_on:
      - xxx-demo
    networks:
      - my-network
networks:
  my-network:
    name: my-network

    # 3.2 部署

    进入yaml文件所在目录，执行以下命令：

    docker-compose up -d
    1

    查看是否部署成功：

    docker images
    1

    默认的镜像名称为root-xxx-demo
