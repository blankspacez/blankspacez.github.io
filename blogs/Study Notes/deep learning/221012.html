    # CNN的基本应用

    CNN大量应用于计算机视觉（computer vision）领域，应用场景包括图像分类、检索、检测、分割等，具体应用包括人脸识别、图像生成、自动驾驶等。

    # 卷积（Convolutional Layer）

    Q：卷积是什么?
    A：convolution is an operation on two functions of a real- valued argument.
    卷积是对两个实变函数的一种数学操作。

    卷积的优势：通过感受野和权值共享减少了神经网络需要训练的参数的个数。

    # 一维卷积

    • 一维卷积经常用在信号处理中，用于计算信号的延迟累积
    • 假设一个信号发生器在时刻t发出一个信号xtx_tx​t​​，其信息的衰减率 fkf_kf​k​​，即在k-1个时间步长后，信息衰减为原来的fkf_kf​k​​倍
    • 设f1f_1f​1​​= 1,f2f_2f​2​​=1/2,f3f_3f​3​​= 1/4,在时刻t收到的信号yty_ty​t​​为当前时刻产生的信息和以前时刻延迟信息的叠加

    yt=1×xt+1/2×xt−1+1/4×xt−2=f1×xt+f2×xt−1+f3×xt−2=∑k=13fk×xt−k+1\begin{aligned} y_t&=1 \times x_t+1/2 \times x_{t-1}+1/4 \times x_{t-2}\\ &=f_1 \times x_t+f_2 \times x_{t-1}+f_3 \times x_{t-2}\\ &=\sum_{k=1}^3f_k \times x_{t-k+1} \end{aligned} ​y​t​​​​​​​=1×x​t​​+1/2×x​t−1​​+1/4×x​t−2​​​=f​1​​×x​t​​+f​2​​×x​t−1​​+f​3​​×x​t−2​​​=​k=1​∑​3​​f​k​​×x​t−k+1​​​​

    • 此处的f=[f1f_1f​1​​,f2f_2f​2​​,f3f_3f​3​​]被称为滤波器(filter) 或卷积核(convolutional kernel)
    • 设滤波器f长度为m,它和一个信号序列 x=[x1x_1x​1​​,x2x_2x​2​​,x3x_3x​3​​...]的卷积记为 yt=∑k=1mfk×xt−k+1y_t=\sum_{k=1}^mf_k \times x_{t-k+1}y​t​​=∑​k=1​m​​f​k​​×x​t−k+1​​

    # 二维卷积

    在图像处理中，图像是以二维矩阵的形式输入到神经网络的，因此我们需要二维卷积。

    二维卷积采用内积的计算方式，如下图：

    Some concept names (以上图为例) :
    input: 3 × 3
    kernal/filter: 2 × 2
    步长 stride: 1
    特征图 feature map1: 2 × 2 feature map2: 1 × 1 若使用该步长发现图像大小不匹配则需引入padding，对图片进行0填充。padding=1表示为矩阵填上一圈的0。
    特征图大小 =（input+2*padding-kernal）/stride +1

    # 池化（Pooling Layer）

    • 类似于缩放过程，保留了主要特征的同时减少参数和计算量，防止过拟合，提高模型泛化能力。

    • 它一般处于卷积层与卷积层之间，全连接层与全连接层之间。

    Pooling的类型:

    • Max pooling：最大值池化，分类任务倾向于最大值池化。

    • Average pooling：平均池化

    # 全连接（Fully Connected Layer）

    • 两层之间所有神经元都有权重链接

    • 通常全连接层在卷积神经网络尾部

    • 全连接层参数量通常最大

    全连接层的主要作用就是将前层（卷积、池化等层）计算得到的特征空间映射样本标记空间。简单的说就是将特征表示整合成一个值，其优点在于减少特征位置对于分类结果的影响，提高了整个网络的鲁棒性。

    # CNN典型结构

    # AlexNet

    AlexNet是具有历史意义的一个网络结构（8层），在AlexNet之前， 深度学习已经沉寂了很久。

    历史的转折在2012年到来，AlexNet 在当年的ImageNet图像分类竞赛中，top-5错误率比上一年的冠军下降了十个百分点，而且远远超过当年的第二名。

    AlexNet之所以能够成功，深度学习之所以能够重回历史舞台，原因在于:

    • 大数据训练:百万级lmageNet图像数据
    • 非线性激活函数: ReLU
    • 防止过拟合: Dropout（随机失活，训练时随机关闭部分神经元,测试时整合所有神经元），Data augmentation（数据增强，1.平移、 翻转、对称，2.改变RGB通道强度）
    • 其他:双GPU实现

    # ZFNet

    2013年ImageNet图像分类竞赛的冠军，ImageNet top 5 error: 16.4% -> 14.8%

    • 网络结构与AlexNet相同
    • 将卷积层1中的感受野大小由 11 x 11改为7 x 7步长由4改为2
    • 卷积层3，4，5中的滤波器个数由384，384， 256改为512， 512， 1024

    # VGG

    VGG是一个更深网络，贡献很大，至今仍常用。

    • 8 layers (AlexNet) -> VGG16 - VGG19
    • ILSVRC top 5错误率从11.7% -> 7.3%（当年亚军，冠军是GoogleNet）

    # GoogleNet

    网络总体结构:

    • 网络包含22个带参数的层(如果考虑pooling层就是27层)，独立成块的层总共有约有100个
    • 参数量大概是Alexnet的1/12
    • 没有FC层

    # ResNet

    贡献很大，至今仍常用

    • 残差学习网络(deep residual learning network)
    • 2015年IL .SVRC竞赛冠军，top5错误率从6.7% -> 3.57%
    • 深度有152层
